## Board Spec Contents

Each board spec file includes:

- **Module / SoC** — chip variant, revision, and certified RF module if applicable
- **Pin assignment table** — GPIO number → peripheral mapping (I2C, SPI, UART, PWM, ADC, DAC, touch, USB)
- **Strapping and reserved pins** — boot-mode pins, JTAG, internal flash lines
- **Power supply rails** — 3.3 V regulator limits, LiPo charger presence, USB VBUS handling
- **Flash and PSRAM** — type (NOR/OPI), size, interface speed, XIP capability
- **Antenna** — PCB trace, IPEX/U.FL connector, ceramic, or integrated module antenna
- **Special notes** — LED/button GPIOs, SD card conflicts, USB CDC considerations
