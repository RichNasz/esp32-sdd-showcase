# Repo Overview Specification

## Purpose
Comprehensive showcase of Spec-Driven Development (SDD) for ESP32 using ESP-IDF.  
Every line of code and every piece of documentation is generated exclusively by AI agents from human-written specifications.

## Key Features
- AI-generated board specs from vendor docs (Adafruit, Seeed, Espressif, LilyGo, etc.)
- 8 progressive examples leveraging ESP32 hardware superpowers
- Full suite of published Agent Skills
- Strict 100 % AI-first workflow

## Quick Start
1. git clone this repo
2. npx @agentskills/cli install RichNasz/esp32-sdd-showcase#skills
3. Edit any FunctionalSpec.md → activate full-project-generator skill

Full documentation → [docs/README.md](docs/README.md)
